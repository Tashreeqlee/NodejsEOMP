Users Table

Http Methods Path Description
POST /login User login.
GET /users Display a list of users.
GET /:id Display a single user.
PUT/PATCH /update/:id Updates or modifies user's record.
POST /register To register a new user.
DELETE /delete/:id To delete a single user.

Products Table

Http Methods Path Description
GET /products Display a list of products.
GET /:id Display a single product.
PUT /update/:id Update a product.
DELETE /delete/:id Delete a product.
POST /addProduct To add new product.
